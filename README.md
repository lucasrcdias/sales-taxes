# Sales taxes

## Requirements

- NodeJS (v10.x)
- Yarn (v1.x)

## Installing

```
git clone git@github.com:lucasrcdias/sales-taxes.git
yarn install
```

## Running

```bash
yarn start input1.txt
yarn start input2.txt
yarn start input3.txt
```

### More inputs

Create a new .txt file and call `yarn start [your new file]`

## Testing

```
yarn test
```
